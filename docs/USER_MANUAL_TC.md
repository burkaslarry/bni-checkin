## 1. 系統簡介

EventXP for BNI Anchor 是一套專為 BNI 分會設計的活動簽到管理系統，提供：

- QR Code 自助簽到
- 即時出席報告（WebSocket 即時更新）
- 會員及嘉賓管理
- CSV 批量匯入/匯出
- 管理員批量簽到
- VIP / 講者角色標記
- 準時 / 遲到自動判定
- PWA 支援（可加到主畫面）

### 系統架構

| 元件 | 說明 |
|------|------|
| 前端 | React + Vite（部署於 Vercel） |
| 後端 | Spring Boot + Kotlin（部署於 Render） |
| 資料庫 | PostgreSQL |

## 2. 快速開始

### 一般使用者（會員/嘉賓）

1. 掃描主辦方分享的 QR Code
2. 選擇「會員」或「嘉賓」
3. 搜尋並選擇自己的名字
4. 按「確認簽到」

### 管理員

1. 開啟管理頁面 `/admin`
2. 建立本週活動（新增活動和二維碼）
3. 匯入嘉賓名單（如需要）
4. 分享 QR Code 給與會者
5. 監控即時報告

## 3. 活動管理

### 建立活動

路徑：管理頁面 → **新增活動和二維碼**

| 欄位 | 說明 | 預設值 |
|------|------|--------|
| 活動名稱 | 例如「Weekly Meeting」 | EventXP for BNI Anchor |
| 活動日期 | 活動舉行的日期 | 今天 |
| 登記開始時間 | 可以開始簽到的時間 | 06:30 |
| 活動開始時間 | 自動設定為登記時間 +30 分鐘 | 07:00 |
| 準時截止 | 超過此時間簽到視為遲到 | 07:05 |
| 活動結束時間 | 自動計算 | 09:00 |

按下「建立活動」後：
- 系統自動將所有會員設為「缺席」
- 自動生成 QR Code
- 自動下載包含 QR Code 的 PDF

### 查看/刪除活動

路徑：管理頁面 → **活動管理**

顯示目前活動資訊，包括活動 ID、日期、各時間設定。

操作：
- **查看即時報告** — 開啟報告頁面
- **重新整理** — 刷新活動資料
- **刪除此活動** — 刪除活動及所有簽到記錄（需確認）

## 4. QR Code 生成與分享

建立活動後，系統自動生成 QR Code。可透過以下方式分享：

| 操作 | 說明 |
|------|------|
| 下載 PDF | 包含活動資訊和 QR Code 的完整文件 |
| 下載 PNG | 單獨的 QR Code 圖片 |
| 複製字串 | 複製 QR Code 內含的連結 |
| WhatsApp 分享 | 直接開啟 WhatsApp 傳送 |
| Email 分享 | 開啟郵件客戶端傳送 |

QR Code 內容為簽到頁面的連結，格式：`https://your-domain/?event=YYYY-MM-DD`

## 5. 會員管理

路徑：管理頁面 → **會員管理** 或 `/admin/members`

### 會員列表

顯示所有會員，包含：
- 姓名
- 專業領域
- 狀態（Standing）

### 會員狀態

| 狀態 | 顏色 | 說明 |
|------|------|------|
| GREEN（正常） | 🟢 | 正常出席 |
| YELLOW（觀察） | 🟡 | 出席率待改善 |
| RED（停權） | 🔴 | 暫停權限 |
| BLACK（已離會） | ⚫ | 已離開分會 |

### 操作

- **編輯** — 修改專業領域及狀態
- **刪除** — 移除會員（需確認）
- **重新載入** — 重新從資料庫載入

---

## 6. 嘉賓管理

路徑：管理頁面 → **嘉賓管理** 或 `/admin/guests`

### 嘉賓列表

顯示所有已登記嘉賓，可按活動日期篩選。

欄位：
- 姓名
- 專業領域
- 邀請人（哪位會員邀請）
- 活動日期

### 操作

- **按活動篩選** — 下拉選單選擇活動日期
- **編輯** — 修改嘉賓資料
- **刪除** — 移除嘉賓（需確認）

---

## 7. 批量匯入

路徑：管理頁面 → **批量匯入** 或 `/admin/import`

### 匯入類型

- **匯入會員** — 批量新增/更新會員
- **匯入嘉賓** — 批量新增/更新嘉賓

### 操作步驟

1. 選擇匯入類型（會員/嘉賓）
2. 下載 CSV 範本
3. 填入資料
4. 上傳 CSV 檔案（拖放或點擊上傳）
5. 預覽解析結果，檢查錯誤
6. 確認後按「開始匯入」

### CSV 格式

**會員 CSV：**
```
name,profession
Alice Chan,室內設計師
Bob Wong,軟件工程師
```

**嘉賓 CSV：**
```
name,profession,phone,referrer,event_date
Alice Chan,室內設計師,91234567,Larry Lo,2026-03-02
```

## 8. 簽到流程

### 自助簽到（與會者使用）

1. 掃描 QR Code 或開啟簽到連結
2. 頁面顯示「EventXP for BNI Anchor 簽到 Check-in」
3. 選擇身份：**會員** 或 **嘉賓**
4. 在搜尋欄輸入名字或專業領域
5. 從列表中選擇自己
6. 按「確認簽到」
7. 顯示簽到成功訊息

### 簽到規則

| 簽到時間 | 狀態 |
|----------|------|
| 準時截止時間之前 | ✅ 準時（on-time） |
| 準時截止時間之後 | ⚠️ 遲到（late） |

### 重複簽到

- 會員：更新簽到時間（資料庫覆蓋）
- 嘉賓：顯示「已經簽到過了」，不允許重複

## 9. 管理員手動簽到

路徑：管理頁面 → **手動輸入**

前提：本週必須已建立活動。

### 單筆簽到

1. 輸入姓名和專業領域
2. 選擇身份（會員/嘉賓）
3. 嘉賓可選擇角色：一般 / VIP / 講者
4. 可選填邀請人
5. 可調整簽到時間
6. 按「確認新增」

### 批量簽到

1. 設定簽到時間
2. 從列表中勾選要簽到的人（分會員和嘉賓兩欄）
3. 可使用快捷按鈕：全選 / 全選會員 / 全選嘉賓 / 清除
4. 按「批量簽到」

## 10. 即時出席報告

路徑：`/report` 或任何頁面點擊「即時報告」

### 出席報告 Tab

即時顯示活動出席狀況：

**統計卡片：**
- 總出席人數
- 準時
- 遲到
- 缺席
- VIP 到場
- 嘉賓人數

**篩選器：** 全部 / 會員 / 嘉賓 / VIP

**出席列表：**
- 姓名、角色標記（VIP/嘉賓/講者）
- 簽到時間
- 遲到標記

**缺席列表：**
- 未簽到的會員名單

### 即時更新

- WebSocket 連線（即時推送更新）
- 每 10 秒自動輪詢作為備份
- 狀態指示：「即時連線中」/「重新連線中」

## 11. 簽到記錄與 CSV 匯出

路徑：報告頁面 → **簽到記錄 CSV** Tab

### 記錄列表

| 欄位 | 說明 |
|------|------|
| # | 序號 |
| 姓名 | 簽到者姓名 |
| 專業領域 | 職業/領域 |
| 類型 | 👤 會員 / 🎫 來賓 |
| 簽到時間 | HKT 本地時間 |
| 操作 | 刪除（僅限嘉賓記錄） |

### 功能

- **搜尋** — 按姓名搜尋
- **篩選** — 全部 / 會員 / 來賓
- **匯出 CSV** — 自訂檔名，下載完整出席記錄
- **清除全部記錄** — 清除所有簽到記錄（需二次確認）

### CSV 匯出內容

```
姓名,專業領域,類別,出席狀態,簽到時間
Larry Lo,客戶服務系統,member,準時,22:49:10
...
Alice Chan,室內設計師,guest,遲到,22:50:05
Bob Wong,軟件工程師,vip,遲到,22:50:20
```

包含：
- 所有已簽到會員（準時/遲到 + 簽到時間）
- 所有缺席會員（缺席，無簽到時間）
- 所有嘉賓/VIP/講者（角色 + 狀態 + 簽到時間）

## 12. Strategic Seating 嘉賓配對

路徑：管理頁面 → **Strategic Seating**

AI 輔助嘉賓與會員配對功能：

1. 輸入嘉賓姓名、專業領域
2. 可選填目標專業、瓶頸、備註
3. 系統自動匹配最適合的會員
4. 顯示配對強度（高/中/低）
5. 支援批量匹配（CSV 上傳）

## 13. 常見問題

### Q: 簽到時顯示「尚未建立活動」？
**A:** 管理員尚未為今天建立活動。請聯絡管理員在管理頁面建立活動。

### Q: 搜尋不到自己的名字？
**A:** 確認選擇了正確的身份（會員/嘉賓）。如果是嘉賓，確認管理員已登記你的名字。

### Q: 掃描 QR Code 後頁面空白？
**A:** 確認手機有網路連線。嘗試刷新頁面或直接開啟網址。

### Q: 如何匯出出席記錄？
**A:** 開啟報告頁面 → 切換到「簽到記錄 CSV」分頁 → 輸入檔名 → 按「匯出 CSV」。

### Q: 可以修改簽到時間嗎？
**A:** 管理員可透過「手動輸入」功能自訂簽到時間重新簽到。

### Q: 嘉賓和 VIP 有什麼分別？
**A:** 嘉賓為一般來賓，VIP 為特別嘉賓。在報告中會以不同標記顯示，CSV 匯出時也會區分角色。

### Q: 系統支援離線使用嗎？
**A:** 系統為 PWA 應用，可加到手機主畫面。但簽到需要網路連線才能記錄到伺服器。

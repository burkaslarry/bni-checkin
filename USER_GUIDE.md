## 👤 會員簽到 Member Check-in

### 步驟 1：掃描活動 QR 碼

1. 點擊首頁的「👤 會員簽到」
2. 允許相機權限
3. 將活動 QR 碼對準相機
4. 點擊「📷 掃描 QR 碼」按鈕

**掃描成功後**：
- 顯示綠色提示：「✅ 活動確認: BNI Anchor Meeting (日期)」
- 活動資訊會顯示在頁面上方

**掃描失敗時**：
- 顯示警告對話框
- 點擊「確定」後可手動選擇會員

### 步驟 2：選擇會員

1. 從下拉選單選擇您的姓名
2. 選單格式：`姓名 - 專業領域`
3. 例如：`Larry Lo - 客戶服務系統`

### 步驟 3：確認簽到

1. 確認預覽資訊正確
2. 點擊「✅ 確認簽到」
3. 看到「✅ [姓名] 簽到成功！」表示完成

## 🎫 來賓簽到 Guest Check-in

### 步驟 1：輸入資料

1. 點擊首頁的「🎫 來賓簽到」
2. 填寫以下資料：

| 欄位 | 說明 | 範例 |
|------|------|------|
| **來賓姓名** | 您的全名 | John Smith |
| **專業領域** | 您的行業/服務 | 網頁設計 |

### 步驟 2：確認簽到

1. 確認預覽資訊正確
2. 點擊「✅ 確認簽到」
3. 看到成功訊息表示完成

**注意**：兩個欄位都必須填寫才能提交

## 🔳 產生 QR 碼 Generate QR Code

### 使用步驟

1. 進入管理後台
2. 點擊「🔳 產生 QR 碼」
3. 填寫活動資訊：
   - **活動名稱**：例如 `BNI Anchor Meeting`
   - **活動日期**：選擇日期
4. QR 碼會自動產生
5. 可下載或直接展示給會員掃描

### QR 碼格式

```json
{
  "eventName": "BNI Anchor Meeting",
  "eventDate": "2025-11-30"
}
```

## 📋 簽到記錄 & 匯出 Records & Export

### 查看記錄

1. 進入管理後台
2. 點擊「📋 簽到記錄 & 匯出」
3. 可查看所有簽到資料

### 功能說明

| 功能 | 說明 |
|------|------|
| **搜尋** | 輸入姓名搜尋特定記錄 |
| **篩選** | 按「全部」「會員」「來賓」篩選 |
| **刷新** | 點擊「🔄 重新整理」更新資料 |
| **刪除** | 點擊 🗑️ 刪除單筆記錄 |
| **匯出 CSV** | 點擊「📥 匯出 CSV」下載檔案 |
| **清除全部** | 點擊「🗑️ 清除全部記錄」清空資料 |

### CSV 匯出格式

| 欄位 | 說明 |
|------|------|
| 姓名 | 簽到者姓名 |
| 專業領域 | 行業/服務 |
| 類別 | member 或 guest |
| Check-in Time | 簽到時間 |
| Server Received Time | 伺服器接收時間 |

## ✍️ 手動輸入 Manual Entry

管理員可直接新增簽到記錄。

### 使用步驟

1. 進入管理後台
2. 點擊「✍️ 手動輸入」
3. 填寫資料：

| 欄位 | 說明 |
|------|------|
| **姓名** | 必填 |
| **專業領域** | 必填 |
| **🎫 嘉賓** | 勾選 = 嘉賓，不勾選 = 會員 |

4. 確認預覽資訊
5. 點擊「✅ 確認新增」

## ❓ 常見問題 FAQ

### Q1: QR 掃描失敗怎麼辦？

**A**: 系統會顯示警告對話框，點擊「確定」後可從下拉選單手動選擇會員。

### Q2: 可以重複簽到嗎？

**A**: 不可以。系統會檢查重複，如果已簽到會顯示「[姓名] 已經簽到過了」。

### Q3: 如何刪除錯誤的記錄？

**A**: 在「📋 簽到記錄 & 匯出」頁面，點擊該記錄旁的 🗑️ 按鈕。

### Q4: CSV 檔案在 Excel 顯示亂碼？

**A**: 系統已加入 UTF-8 BOM，正常情況下不會亂碼。如仍有問題，請在 Excel 中選擇「UTF-8」編碼開啟。

### Q5: 會員名單如何更新？

**A**: 請聯繫系統管理員更新後端的 `members.csv` 檔案。

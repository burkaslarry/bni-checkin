# 📝 EventXP 使用指南（廣東話版）

> BNI Anchor Check-in System 用戶手冊

---

## 系統簡介

EventXP 係專為 BNI Chapter 設計嘅 QR Code 簽到系統，幫你：
- 自動化活動管理
- 即時同步出席資料
- 簡化後續跟進流程

---

## 🔗 快速連結

| 功能 | 連結 | 邊個用 |
|------|------|--------|
| 📱 簽到頁面 | https://bni-anchor-checkin.vercel.app | 會員／嘉賓掃 QR Code 後用 |
| 📊 報表頁 | https://bni-anchor-checkin.vercel.app/report | 副主席／秘書團隊 |
| 🔧 管理後台 | https://bni-anchor-checkin.vercel.app/admin | 限管理員（Larry 等） |

---

## 🔄 現場簽到流程

```
會員／嘉賓到場 → 掃 QR Code → 開簽到網站 → 選擇自己／輸入資料 → 完成 ✅
```

---

## 📋 第一步：活動初始化（管理員適用）

活動開始前，管理員要先設定簽到系統：

### 步驟：
1. **進入後台**：去 https://bni-anchor-checkin.vercel.app/admin
2. **產生活動**：撳「🔳 新增活動和二維碼」
3. **時間設定**：輸入「登記開始時間」，系統會自動計算其他時段
4. **準備現場物資**：
   - 撳「建立活動」
   - 然後撳「下載 PDF」
   - 將 PDF 列印出嚟，放喺接待處俾賓客掃描

💡 **貼士**：建議將簽到網址加入手機瀏覽器書籤，方便隨時調用。

---

## 📱 第二步：現場簽到流程

### 會員簽到：
1. 掃描 PDF 上嘅 QR Code（或直接去 https://bni-anchor-checkin.vercel.app）
2. 選擇「會員」
3. 喺下拉選單搵自己個名
4. 撳「簽到」

### 嘉賓簽到：
1. 掃描 QR Code
2. 選擇「來賓」
3. 可以揀：
   - **📋 已登記來賓** - 喺下拉選單選擇（如果已預先登記）
   - **✍️ 手動輸入** - 自己填姓名同專業領域
4. 撳「簽到」

---

## 📊 第三步：即時監控（副主席／秘書團隊）

活動期間，可以隨時去報表頁睇情況：

**網址**：https://bni-anchor-checkin.vercel.app/report

**可以睇到**：
- 即時出席人數
- 會員 vs 嘉賓比例
- 嘉賓來源（邊個會員帶嚟）
- 專業領域分佈

---

## ✍️ 第四步：手動補錄（後勤支援）

如果嘉賓唔方便自己掃碼，行政團隊可以幫手：

### 步驟：
1. 進入管理頁面：https://bni-anchor-checkin.vercel.app/admin
2. 撳「✍️ 手動輸入」
3. 填寫資料：
   - 嘉賓姓名
   - 專業領域（最重要！）
   - 邀請人（邊位會員帶佢嚟）
4. 撳「確認」

⚠️ **重要**：專業領域 (Domain) 係 BNI 媒合嘅核心，一定要填準確！

---

## 📋 第五步：活動總結（數據導出）

活動完咗之後：

1. 去管理員頁面：https://bni-anchor-checkin.vercel.app/admin
2. 撳「📋 簽到記錄 & 匯出」
3. 揀日期，撳「匯出 CSV」
4. 可以用嚟：
   - 匯入 BNI Connect
   - 安排 121 跟進
   - 分析嘉賓轉化率

---

## 💡 最佳實踐

### 🎯 專業領域係核心！
- Domain 係 BNI 媒合嘅核心
- 手動輸入時一定要填準確
- 可以大大提升後續業務對接效率

### 📱 加入書籤
- 建議將簽到網址加入手機書籤
- 方便隨時調用

### 🕒 提前準備
- 建議活動前 1 日產生 QR Code
- 列印出嚟放喺接待處
- 測試吓系統係咪正常運作
